Key Value Notes
ida:GraphResourceId https://graph.windows.net Azure GraphApi standard resource URL.
ida:GraphUserUrl https://graph.windows.net/{0}/me?api-version=2013-11-08 Azure GraphApi resource URL for current logged on user. It's the latest version as of this day so may change in future.
todo:TodoListResourceid https://{tenantName}/raft-adfs-service-03 App Id URI of the AAD app associated with web api app.
todo:TodoListBaseAddress https://raftsampleservice2.azurewebsites.net URL of Web Api hosted on Azure
ida:ClientId {guid} Application ID of AAD app (raft-adfs-test-02) associated with web app
ida:AppKey {secretHash} Key (generated from Azure portal) associated with raft-adfs-test-02 AAD app.
ida:Tenant {tenantName} FQDN of Azure AD tenant
ida:AADInstance https://login.microsoftonline.com/{0} Constructed at run time in application startup.
ida:RedirectUri https://raftadfstest2.azurewebsites.net/ Reply URL configured for raft-adfs-test-02 AAD app. Don't ignore trailing slash at end. Must match with what configured from portal.
ida:RedisEndpoint raftCache.redis.cache.windows.net Host name of redis cache used
ida:RedisAccessKey {hash} Primary access key associated with redis cache instance used.
ida:CertName CN=sqlAuthTestToken Name of (self-signed) certificate uploaded to web app. This is needed if we are not using password for SPN but certificate. This SPN is associated with Raft-ADFS-Sql-03 AAD app for SQL authentication. So either use this or ida:ClientSecretSqlAadApp key.
ida:ClientIdSqlAadApp {guid} Application ID of AAD app (Raft-ADFS-Sql-03) created for SQL authentication via SPN mode.
ida:ClientSecretSqlAadApp {password} Secret key associated with SPN of AAD app (Raft-ADFS-Sql-03) created for SQL authentication.
sqldb:ResourceId https://database.windows.net/ Azure SQL DB resource URL. It's standard.
